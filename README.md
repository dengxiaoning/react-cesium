# react-cesium

Case project using cesium_dev_kit in react

# install dependencies

```
 yarn
```

# test

```
 yarn run test
```

# dev

```
 yarn run start
```

# build

```

yarn run build

```

# Preview Address

```
http://localhost:3000

```
